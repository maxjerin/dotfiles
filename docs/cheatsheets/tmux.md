### Window Operations

* create new window
`C-b :neww`

* rename window
`C-b ,`

* switch window
`C-b p`: previous
`C-b n`: next:

### Session Operations

* list sessions
`tmux list-sessions`

* rename current session
`C-b $`

* attach to a session
`tmux attach -s <session-name>`


### Pane Operations

* resize pane
`C-b Ctrl+<arrow>`

* switch pane
`C-b <arrow>`
